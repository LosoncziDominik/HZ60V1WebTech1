Git parancsok:

  • git init: inicializálja a git rendszert, a jelenlegi könyvtárba és így létrehoz egy új repozitorit
  • git clone: ezzel a paranccsal klónozni lehet egy git repót a saját gépre
  • git add: ez a parancs egy megadott file-t, feljészíti a commit-ra
  • git commit: ez a parancs commit-olja, az összes olyan file-t, amit a "git add" paranccsal hozzáadtunk. Lehet szöveges üzenetet írn commit előtt.
  • git push: ez a parancs feltölti a commit-olt változásokat a távoli repoba
  • git pull: ezzel a paranccsal lehet frissíteni a helyi repot
